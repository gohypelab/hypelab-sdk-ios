What ships today

SDK Capabilities

Rewarded ads

Load and present rewarded ad placements with callback hooks for readiness, reward, playback, impressions, clicks, and errors.

Native iOS integration

Use the same binary framework from Swift or Objective-C apps with public headers included in the package.

Release-ready packaging

Includes an XCFramework for CocoaPods and a zipped XCFramework target for Swift Package Manager.

Attribution readiness

SKAdNetwork Participation

HypeLab is preparing to register as an Apple SKAdNetwork ad network participant. Apple requires ad networks to register before using SKAdNetwork attribution APIs, receive an ad network identifier, provide a public key, and configure an install-validation postback endpoint.

After Apple issues HypeLab's ad network identifier, app developers who display HypeLab ads will add the lowercase identifier to their app's Info.plist.

Developer setup

<key>SKAdNetworkItems</key>
<array>
  <dict>
    <key>SKAdNetworkIdentifier</key>
    <string>YOUR_HYPELAB_SKADNETWORK_ID.skadnetwork</string>
  </dict>
</array>

Replace the placeholder with the official HypeLab SKAdNetwork ID after registration is complete.

Developers

  • Add HypeLab's issued SKAdNetwork identifier to Info.plist.
  • Use the current SDK release from this repository.
  • Keep the identifier lowercase so iOS recognizes it.

HypeLab

  • Maintain the Apple-issued ad network identifier.
  • Sign eligible ads with the registered private key.
  • Receive and validate SKAdNetwork install-validation postbacks.

Installation

Add HypeLab SDK

Swift Package Manager

Add this repository URL in Xcode under File > Add Package Dependencies.

https://github.com/gohypelab/hypelab-sdk-ios

CocoaPods

Add the pod to your Podfile, then run pod install.

pod 'HypeLabSdk', '~> 1.0'

Rewarded placements

Integrate Rewarded Ads

Swift

import HypeLabSdk

let rewarded = Rewarded(
  placementSlug: "your-placement-slug"
)

rewarded.onReady = {
  print("Rewarded ad is ready")
}

rewarded.onReward = {
  print("Reward the user")
}

rewarded.onError = {
  print("Rewarded ad failed to load")
}

rewarded.show()

Objective-C

#import <HypeLabSdk/Rewarded.h>

Rewarded *rewarded = [[Rewarded alloc]
  initWithPlacementSlug:@"your-placement-slug"];

rewarded.onReady = ^{
  NSLog(@"Rewarded ad is ready");
};

rewarded.onReward = ^{
  NSLog(@"Reward the user");
};

rewarded.onError = ^{
  NSLog(@"Rewarded ad failed to load");
};

[rewarded show];
